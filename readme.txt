Easily manage wholesale products in your WooCommerce shop.

== Description ==
Wholesale Manager for WooCommerce is the most powerful WooCommerce B2B plugin created for WooCommerce store owners. Sell products at wholesale prices to your registered B2B customers and also sell products at regular prices to your B2C customers all within your WooCommerce store.

Wholesale Manager for WooCommerce plugin bridges this gap by seamlessly integrating all the necessary functionalities. Transform your WooCommerce store into a complete B2B wholesale store without any compromise.

=== ✨ Wholesale Manager for WooCommerce Features: ===

- **Wholesale Pricing Options:** Ability to set different wholesale pricing for products, either as a fixed amount or a percentage discount based on user roles or quantity purchased.
- **Tiered Pricing Structures:** Ability to set different pricing levels for different groups of customers (like retailers, distributors, etc.), enabling dynamic pricing based on volume or customer type.
- **Role-based Discount on Product Categories:** Ability to make customized categories for wholesale customers on each role, potentially with different pricing or product offerings.
- **Role-based Discount on Products:** Ability to make customized products for wholesale customers on each role, potentially with different pricing or product offerings.
- **Minimum Order Quantities:** Set minimum order quantities for wholesale orders, ensuring that wholesale customers meet certain volume requirements.
- **Wholesale Registration and Management:** Allow for the creation of wholesale customer accounts with custom registration fields, approval processes, and role assignments.
- **Customize Default Email Template:** Change the default email templates that will be used for notifying registering New Wholesale User, Wholesale User Approved, Declined Wholesale User, and Pending Wholesale User emails.
- **Custom User Roles and Permissions:** Create custom user roles for wholesale customers with specific permissions and access rights, differentiating them from regular retail customers.
- **Private Store or Exclusive Sections:** Create private sections of the store accessible only to approved wholesale customers or even a completely private store for wholesale transactions.
- **Product Visibility Control:** Control which products are visible to wholesale customers and retail customers, allowing for a segmented product catalog.
- **Customizable Shipping and Payment Options:** Ability to disable default payment gateways and shipping methods to provide tailored shipping and payment options for wholesale orders.
- **Customizable Tax and Coupons:** Ability to disable default tax collection and coupons for wholesale users.
- **Multi-currency and Multi-language Support:** Support for multiple currencies and languages to cater to a global wholesale customer base.
- **Integration with Existing WooCommerce Features:** Ensure seamless integration with existing WooCommerce features and plugins, maintaining a consistent user experience and functionality.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/wc-wholesale-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WooCommerce->Settings->Wholesale Manager screen to configure the plugin
4. That's it. You're ready to go!

= Minimum requirements =

* WordPress 5.0 or greater
* WooCommerce 3.0 or greater
* PHP version 7.4 or greater
* MySQL version 5.6 or greater
* MariaDB version 10 or later

###Automatic Install From WordPress Dashboard

1. Login to your the admin panel
2. Navigate to Plugins -> Add New
3. Search **WooCommerce Category Showcase**
4. Click install and activate respectively.

###Manual Install From WordPress Dashboard

If your server is not connected to the Internet, then you can use this method-

1. Download the plugin by clicking on the red button above. A ZIP file will be downloaded.
2. Login to your siteâ€™s admin panel and navigate to Plugins -> Add New -> Upload.
3. Click choose file, select the plugin file and click install

###Install Using FTP

If you are unable to use any of the methods due to internet connectivity and file permission issues, then you can use this method-

1. Download the plugin by clicking on the red button above.A ZIP file will be downloaded.
2. Unzip the file.
3. Launch your favorite FTP client. Such as FileZilla, FireFTP, CyberDuck etc. If you are a more advanced user, then you can use SSH too.
4. Upload the folder to wp-content/plugins/
5. Log in to your WordPress dashboard.
6. Navigate to Plugins -> Installed
7. Activate the plugin
